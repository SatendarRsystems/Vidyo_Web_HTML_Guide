The following points need to understand: 
1)  Download the code
2)	Update the app_id, api_key at file js/tokengenerator.js.
    Steps to get the app_id and api_key : 
    1. Sign in www.vidyo.io
    2. go to user -> "API Keys"
3)  Run index.html file for setup video. 

Note : 
1)  We do not have any permanent account, so our demo purspose app_id and app_key may expire. 
    Vidyo/User need to update these values (api_key and app_id) at js/tokengenerator.js.
2)	Application will run on Firefox/Chrome. 
3)  Application will run on localhost or https only.
4)  For online IDE rendering, we are using stackblitz SDK.
